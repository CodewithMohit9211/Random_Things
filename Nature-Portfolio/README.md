# Living Forest Portfolio

A breathtaking student portfolio concept built with only HTML, CSS, and JavaScript.

## What it includes
- Cinematic hero section with atmospheric sky, mountains, fog, and glowing particles
- Glassmorphism cards, floating nature visuals, and premium environmental styling
- Interactive day/night mode, rain toggle, and ambient sound control
- Animated skills, project filters, journey timeline, gallery, and contact interface
- Responsive layout for desktop and mobile
- Custom cursor, scroll reveals, and immersive motion transitions

## Files
- `index.html` — main website structure and layout
- `css/style.css` — all styling, animated backgrounds, and responsive design
- `js/main.js` — interactive controls, canvas particles, theme toggles, and scroll effects

## How to open
1. Open `index.html` in your browser.
2. Use the top controls to switch day/night, toggle rain, and enable ambient sound.
3. Scroll through sections to experience the animated storytelling and natural effects.
4. Press `M` or click "Find the hidden path" to reveal the hidden easter egg.

## Notes
- This project is intentionally built without external libraries.
- The ambient sound uses the Web Audio API for a subtle forest texture.
- For best performance, use a modern browser.
